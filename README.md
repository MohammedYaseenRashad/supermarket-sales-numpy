# Supermarket Sales Analysis using NumPy

## Project Overview
This project analyzes supermarket sales data using NumPy to perform numerical analysis and extract basic insights.

## Tools Used
- Python
- NumPy
- Pandas
- seaborn
- Jupyter Notebook

## Analysis Performed
- Data loading and inspection
- Numerical calculations (sum, mean, min, max)
- Simple analysis to understand sales patterns

## Dataset
The dataset used for this project is included in the repository.

## Purpose
This project was created for learning and practice of NumPy in data analysis.
